# 🎯 Number Guessing Game

A simple and interactive **Number Guessing Game** built using **HTML, CSS, and JavaScript**.

The game generates a random number between **1 and 100**, and the player has to guess it. The application provides hints after every guess and automatically starts a new game after a correct guess.

---

## 🌐 Live Demo

🔗 https://rishyandhark-sudo.github.io/Guessing_Number/

---

## 📌 Features

* 🎲 Random number generation (1–100)
* 📉 Displays "Too Low" for smaller guesses
* 📈 Displays "Too High" for larger guesses
* 🎉 Success message when the correct number is guessed
* 🔄 Automatically generates a new number after 3 seconds
* 📝 Tracks the number of attempts
* ⚠️ Validates empty input
* 🎨 Responsive and modern user interface
